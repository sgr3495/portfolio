Audio Visualizer

Fall 2015

Skill

  • Swift
  • iOS8

Project Description

The audio visualizer is a interactive application. It was basically showing the sound wave base on the sound itself. This allow the user to interactive by changing the color on canvas. They also can add or remove the different kind of readymade frequency. It also include three songs which are New Adventure Theme, Peanuts Theme, and The Picard Song.

Project Timeline

Research

This is a solo project. I admit that I don’t know how the sound wave was created. So, I did google it across internet to understand the frequency, pitch, and the loudness. My ear doesn’t really pick up low bass. So it was surprising for me to see that sub bass frequency include harp and tuba. I always thought those are high frequency the whole time. Now, I know. After the fantastic research

Coding Time

I use Web Audio API to analyze the sound which allow me to it. Using AnalyserNode give me an access to the frequency data of the sound. I converted the data into drawing on canvas. I played with it to create cool stuff with it. I also add filter to make it more eye candy.

Conclusion

It was fun to try new stuff using audio on canvas. If there is something I want to do, I will want to learn how to separate the specific instrument, so that way I can try to copy DubstepGutter style. However, I realized that it won’t be possible, but it will be possible if the sound is playing separately and combine them all together to create one awesome drawing of wavelength. I could try make 3d version of sound wave. This project is something that I will love to look back and improve it.
